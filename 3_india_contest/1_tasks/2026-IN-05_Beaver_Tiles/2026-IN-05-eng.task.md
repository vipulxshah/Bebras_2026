## Body

Beaver Kito has a row of 7 tiles. Each tile is Light ![light] on one side and Dark ![dark] on the other side.

In each move, Kito chooses exactly two tiles and flips both of them, so that:
- Light becomes Dark
- Dark becomes Light

Kito can repeat this move as many times as he wants.

The starting pattern of the tiles is shown in the picture.

## Answer Explanation

The correct answer is B.

In each move, Kito flips exactly two tiles. This means:
- Two Light tiles can become Dark
- Two Dark tiles can become Light
- Or one Light and one Dark can swap

So, the total number of Dark tiles always changes by 0 or 2.

Because the change is always an even number, the parity of the Dark tiles (whether the number of Dark tiles is odd or even) never changes.
- Starting Pattern: There are 2 Dark tiles, which is an even number.
- Options A, C, and D: These have 4, 4, and 2 Dark tiles respectively. Since these are all even, they can be reached.
- Option B: This pattern has 3 Dark tiles, which is an odd number. It is impossible to
reach an odd number from an even starting point.


## This is Informatics

This task is about invariants. In informatics, an invariant is something that does not change, even after applying many operations.

Here, each move flips two tiles. Because of this, the number of Dark tiles can change by 0 or 2, but it can never change from even to odd, or odd to even.

Instead of trying all possible moves, we look for such unchanging properties. This helps us decide which situations are possible and which are not, without testing everything.

Invariants are widely used in informatics to:
- check whether algorithms work correctly, 
- prove that certain states cannot be reached, 
- reason about systems with many possible states.

## This is Computational Thinking

To solve this task, students must look beyond individual moves and identify a property that remains unchanged after every operation. Each move flips exactly two tiles, so students need to track how this affects the overall pattern rather than focusing on specific sequences of moves.

Students begin by observing how the number of Dark tiles changes with each move. Instead of trying multiple combinations, they recognize a consistent pattern: the number of Dark tiles changes in pairs. This helps them focus on a global rule rather than exploring all possibilities.

The problem encourages abstraction, as students shift their attention from individual tile changes to a higher-level property (parity of Dark tiles). It also develops pattern recognition, as students identify that the number of Dark tiles always remains either even or odd.

Additionally, students use logical reasoning to explain why certain patterns cannot be reached. By applying this invariant, they eliminate impossible options without relying on trial and error.
