## Body

After a storm, part of the beaver dam has collapsed. Five beavers Asha, Bimal, Chitra, Dev, and Esha rush to carry out repairs.

![taskbody]

Each beaver performs exactly one task, and each task is performed by exactly one beaver. The repair tasks are:

- Stabilizing logs
- Redirecting mud
- Cutting wood
- Fixing joints
- Clearing debris

Each beaver knows how to do two of the tasks, as shown in the table below.

:--- | :---
**Beaver** | **Tasks they can perform**
Asha	| Stabilizing logs, Redirecting mud
Bimal	| Stabilizing logs, Cutting wood
Chitra | Clearing debris, Redirecting mud
Dev	| Fixing joints, Cutting wood
Esha | Clearing debris, Fixing joints

## Answer Options/Interactivity Description

A) Chitra clears debris and Dev cuts wood
B) Esha clears debris and Bimal cuts wood
C) Asha stabilizes logs and Dev fixes joints
D) Esha clears debris and Bimal stabilizes logs


## Answer Explanation

The correct answer is D.

We can examine how the assignments are determined using the example below.

Asha can either stabilize logs or redirect mud.
First, suppose that Asha stabilizes logs. Then the only beaver left to redirect mud is Chitra; the only beaver left to clear debris is Esha; the only beaver left to fix joints is Dev; and the only beaver left to cut wood is Bimal.

Second, suppose that Asha redirects mud instead. Then the only beaver left to stabilize logs is Bimal; the only beaver left to cut wood is Dev; the only beaver left to fix joints is Esha; and the only beaver left to clear debris is Chitra.

There are no other possible assignments of jobs to beavers.

Thus, we can see that:
- answer option A is consistent with the second assignment;
- answer options B and C are consistent with the first assignment;
- answer option D is consistent with neither.

## This is Informatics

This task represents a constraint satisfaction problem (CSP), in computer science.
- There is a fixed set of variables (the beavers). 
- Each variable must be assigned one value from a limited domain (the tasks the beaver can safely do). 
- The assignments must satisfy global constraints: 
  - each task is assigned to exactly one beaver, 
  - each beaver gets exactly one task, 
  - only allowed task-beaver combinations are permitted. 

Such problems arise in computer science in areas like:
- job assignment in operating systems, 
- resource allocation in distributed systems

## This is Computational Thinking

To solve this task, students must work systematically with multiple constraints and narrow down possibilities step by step. Each beaver can perform only certain tasks, and each task must be assigned to exactly one beaver. This requires students to carefully track which options remain valid after each decision.

Students begin by analyzing the given table and identifying limited choices (for example, tasks that only two beavers can perform). They then use elimination: when one beaver is assigned a task, other dependent choices are restricted. This creates a chain of decisions, where each step influences the next.

The problem encourages decomposition, as students break the larger assignment problem into smaller parts (assigning one task at a time). It also develops logical reasoning, since students must justify why certain combinations are possible or impossible.

Additionally, students engage in systematic case analysis, testing each option and checking whether all constraints are satisfied. This mirrors how computational problems are solved by exploring possible states while discarding invalid ones.

Overall, the task builds key computational thinking skills such as constraint handling, step-by-step deduction, and structured problem solving, which are essential when dealing with complex problems involving multiple conditions.
